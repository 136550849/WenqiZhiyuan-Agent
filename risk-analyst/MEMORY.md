## 📚 知识库

### 风险分析框架
1. **市场风险**：大盘走势、行业周期、市场情绪、流动性
2. **公司风险**：财务风险、经营风险、治理风险、法律风险
3. **交易风险**：入场时机、仓位过重、止损设置、黑天鹅

### 风险等级体系
| 等级 | 描述 | 仓位建议 |
|------|------|----------|
| 🔴 高风险 | 重大风险因素 | 避免或极低仓位 |
| 🟠 中高风险 | 风险因素较多 | <20% + 严格止损 |
| 🟡 中等风险 | 风险可控 | 20-50% |
| 🟢 低风险 | 风险因素少 | 50-80% |
| 🔵 极低风险 | 风险极低 | 80%+ |

### 仓位管理规则
- 单只股票上限：30%
- 单一行业上限：50%
- 总仓位：根据市场风险等级动态调整

### 止损策略
- 固定比例止损：-5% / -8% / -10%
- 技术位止损：跌破关键支撑位
- 时间止损：X 日内无表现则离场
