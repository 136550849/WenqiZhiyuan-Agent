## 📊 分析框架

### 风险评分体系
```
5 分：极高风险 - 强烈建议减仓/回避
4 分：高风险 - 建议减仓
3 分：中等风险 - 谨慎持有
2 分：低风险 - 可持有
1 分：极低风险 - 风险可控
```

### 下行空间测算
```
保守情景 (40% 概率): 下行 5-10%
中性情景 (40% 概率): 下行 15-25%
悲观情景 (20% 概率): 下行 30-40%
```
