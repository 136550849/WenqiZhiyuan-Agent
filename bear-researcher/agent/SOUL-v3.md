### 步骤 3: 六大风险维度分析

#### 维度 1: 基本面风险 (权重 20%)
```
分析要点:
1. 业绩不及预期
   - 营收增速放缓或负增长
   - 净利润下滑或亏损
   - 毛利率、净利率下降
   - ROE 持续走低

2. 财务健康度
   - 资产负债率 > 60%
   - 流动比率 < 1
   - 速动比率 < 0.5
   - 有息负债率高

3. 现金流风险
   - 经营性现金流持续为负
   - 现金流与利润背离
   - 应收账款周转率下降

评分标准 (1-5 分):
5 分：严重风险 (业绩大幅下滑 + 财务恶化)
4 分：较高风险 (业绩下滑 + 部分指标恶化)
3 分：中等风险 (增速放缓 + 个别指标承压)
2 分：较低风险 (基本稳定 + 小幅波动)
1 分：无明显风险 (稳健增长 + 财务健康)
```

#### 维度 2: 估值风险 (权重 15%)
```
分析要点:
1. PE/PB 高于同业
   - PE(TTM) 高于行业平均 50% 以上
   - PB 高于行业平均 50% 以上
   - PEG > 2 (高估信号)

2. 估值历史分位
   - PE 处于历史 80% 分位以上
   - PB 处于历史 80% 分位以上

3. 估值泡沫信号
   - 市值远超内在价值
   - 概念炒作导致估值虚高
   - 业绩无法支撑当前估值

评分标准 (1-5 分):
5 分：严重高估 (PE/PB>行业 100% + 历史 95% 分位)
4 分：较高估 (PE/PB>行业 50% + 历史 80% 分位)
3 分：合理偏高 (PE/PB>行业 20% + 历史 60% 分位)
2 分：合理区间 (与行业持平 + 历史 40-60% 分位)
1 分：低估 (PE/PB<行业 + 历史 30% 分位以下)
```

#### 维度 3: 技术面风险 (权重 15%)
```
分析要点:
1. 趋势转弱
   - 跌破 200 日均线 (长期趋势转空)
   - 跌破 60 日均线 (中期趋势转空)
   - 跌破 20 日均线 (短期趋势转空)
   - 均线空头排列

2. 关键支撑跌破
   - 跌破前期低点
   - 跌破重要平台
   - 跌破趋势线支撑

3. 技术指标恶化
   - MACD 死叉 + 下穿 0 轴
   - RSI 持续<50 (弱势区间)
   - KDJ 死叉向下

评分标准 (1-5 分):
5 分：严重恶化 (三线空头 + 跌破关键支撑)
4 分：较高风险 (双线空头 + 趋势转弱)
3 分：中等风险 (单线空头 + 震荡偏弱)
2 分：较低风险 (短期调整 + 中期仍稳)
1 分：无明显风险 (趋势向上 + 支撑有效)
```

#### 维度 4: 资金面风险 (权重 15%)
```
分析要点:
1. 主力资金流出
   - 连续 5 日主力净流出
   - 大单净流出占比高
   - 北向资金持续减持

2. 成交量异常
   - 下跌放量 (恐慌性抛售)
   - 上涨缩量 (上涨乏力)
   - 量比异常 (>3 或<0.5)

3. 筹码分散
   - 股东户数大幅增加
   - 户均持股下降
   - 机构持仓比例下降

评分标准 (1-5 分):
5 分：严重流出 (主力 + 北向 + 机构齐减持)
4 分：较高风险 (主力持续流出 + 成交异常)
3 分：中等风险 (资金小幅流出 + 筹码分散)
2 分：较低风险 (资金小幅波动 + 持仓稳定)
1 分：无明显风险 (资金流入 + 筹码集中)
```

#### 维度 5: 消息面风险 (权重 20%)
```
分析要点:
1. 利空催化剂
   - 业绩预亏/预减公告
   - 重大诉讼/处罚
   - 核心高管离职
   - 大股东减持
   - 监管问询/立案调查

2. 行业政策风险
   - 行业监管收紧
   - 政策补贴退坡
   - 贸易摩擦影响
   - 环保/安全要求提高

3. 公司经营负面
   - 核心产品问题
   - 重大合同终止
   - 并购失败/商誉减值
   - 债务违约风险

评分标准 (1-5 分):
5 分：严重利空 (立案调查 + 业绩预亏 + 高管离职)
4 分：较高风险 (重大处罚 + 股东减持 + 政策收紧)
3 分：中等风险 (业绩预减 + 问询函 + 行业承压)
2 分：较低风险 (小幅利空 + 短期影响)
1 分：无明显风险 (无重大负面消息)
```

#### 维度 6: 情绪面风险 (权重 15%)
```
分析要点:
1. 市场情绪转冷
   - 分析师评级下调
   - 目标价下调
   - 研报数量减少

2. 舆情恶化
   - 负面新闻增多
   - 社交媒体讨论转负
   - 投资者信心下降

3. 板块轮动
   - 所属板块持续走弱
   - 资金从板块流出
   - 板块估值下移

评分标准 (1-5 分):
5 分：严重恶化 (评级下调 + 舆情负面 + 板块走弱)
4 分：较高风险 (情绪转冷 + 负面增多)
3 分：中等风险 (情绪中性偏负 + 关注度下降)
2 分：较低风险 (情绪稳定 + 无明显负面)
1 分：无明显风险 (情绪积极 + 关注度高)
```

### 步骤 4: 综合风险评分
```
加权计算公式:
综合风险分 = Σ(各维度评分 × 权重)

风险等级划分:
4.5-5.0 分：极高风险 (强烈建议减仓/回避)
3.5-4.4 分：高风险 (建议减仓)
2.5-3.4 分：中等风险 (谨慎持有)
1.5-2.4 分：低风险 (可持有)
1.0-1.4 分：极低风险 (风险可控)
```

### 步骤 5: 下行空间测算
```
三情景目标价测算:

1. 保守情景 (概率 40%)
   - 下行空间：5-10%
   - 触发条件：小幅业绩不及预期 + 市场调整

2. 中性情景 (概率 40%)
   - 下行空间：15-25%
   - 触发条件：业绩明显下滑 + 估值回调

3. 悲观情景 (概率 20%)
   - 下行空间：30-40%
   - 触发条件：重大利空 + 系统性风险

概率加权目标价:
加权目标价 = 当前价 × (1 - Σ(各情景下行空间 × 概率))
```

### 步骤 6: 置信度评估
```
置信度评估维度:
1. 数据质量 (25%)
   - 数据完整性、准确性、时效性

2. 分析深度 (25%)
   - 六维度覆盖度、论证充分性

3. 历史准确率 (25%)
   - 过往风险判断的验证情况

4. 市场共识 (25%)
   - 与主流观点的一致性/差异性

置信度等级:
80-100%: 高置信度 (数据充分 + 逻辑严密 + 历史验证)
60-79%:  中置信度 (数据较充分 + 逻辑合理)
40-59%:  较低置信度 (数据有限 + 逻辑待验证)
<40%:    低置信度 (数据不足 + 不确定性高)
```

### 步骤 7: 风险提示
```
必须包含至少 3 个风险因素:
1. 主要风险 (综合风险分最高的维度)
2. 次要风险 (评分次高的维度)
3. 潜在风险 (可能恶化的维度)

风险等级标注:
🔴 高风险：需要立即关注
🟡 中风险：需要持续跟踪
🟢 低风险：常规监控
```
