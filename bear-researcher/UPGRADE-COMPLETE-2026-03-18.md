## 🎯 v3.0 Pro 核心能力

### 1. 六大风险维度分析
```
基本面风险 (20%)  → 业绩/财务/现金流
估值风险 (15%)    → PE/PB/历史分位
技术面风险 (15%)  → 趋势/支撑/指标
资金面风险 (15%)  → 主力/成交量/筹码
消息面风险 (20%)  → 利空/政策/经营
情绪面风险 (15%)  → 评级/舆情/板块
```

### 2. 量化风险评分
```
5 分：极高风险 (强烈建议减仓/回避)
4 分：高风险 (建议减仓)
3 分：中等风险 (谨慎持有)
2 分：低风险 (可持有)
1 分：极低风险 (风险可控)
```

### 3. 三情景下行测算
```
保守情景 (40% 概率): 下行 5-10%
中性情景 (40% 概率): 下行 15-25%
悲观情景 (20% 概率): 下行 30-40%

加权目标价 = 当前价 × (1 - Σ各情景下行空间×概率)
```

### 4. 置信度评估
```
80-100%: 高置信度 (数据充分 + 逻辑严密)
60-79%:  中置信度 (数据较充分 + 逻辑合理)
40-59%:  较低置信度 (数据有限 + 逻辑待验证)
<40%:    低置信度 (数据不足 + 不确定性高)
```
