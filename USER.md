# USER.md - 关于你的交易助手

## 系统信息

- **平台**：TradingAgents-CN v2.0.0
- **框架**：OpenClaw Gateway
- **模型**：Qwen3.5-Plus (阿里云百炼)
- **数据源**：Tushare Pro

## 如何使用

### 快速开始

**单股分析**：
```
分析 000001.SZ 平安银行
```

**指定分析师**：
```
@市场分析师 分析 600519.SH 的技术面
@基本面分析师 分析 300750.SZ 的财务状况
```

**综合分析**：
```
综合分析 000001.SZ，包括技术面、基本面和风险评估
```

### 访问方式

1. **Web 平台**：http://localhost:9981
2. **Control UI**：http://localhost:18789
3. **命令行**：`openclaw agents list`

## 数据说明

### Tushare 数据覆盖

- **A 股**：全部上市公司
- **指数**：主要市场指数
- **基金**：公募基金
- **期货**：主要期货品种
- **宏观经济**：主要经济指标

### 数据更新频率

- **日线数据**：每个交易日 16:00 后更新
- **财务数据**：季报/年报发布后更新
- **新闻数据**：实时更新

## 风险提示

⚠️ **重要声明**：

1. 本系统为**辅助分析工具**，不构成投资建议
2. 所有分析结果基于历史数据，不保证未来表现
3. 投资有风险，决策需谨慎
4. 请结合自身判断和专业顾问意见
