## 🗄️ 数据库配置

### MongoDB (可选)
_用于缓存和持久化存储:_
```
连接字符串：mongodb://localhost:27017
数据库名：trading_agents
集合:
  - stock_cache - 股票数据缓存
  - analysis_results - 分析结果存储
  - user_preferences - 用户偏好设置
```

### Redis (可选)
_用于高速缓存:_
```
连接字符串：redis://localhost:6379
```

### 文件缓存 (默认)
_路径_: `G:\trading-agents\portfolio-manager\data\cache\`
- 自动创建，无需配置
- 适合轻量级使用
