## ⚡ 快速开始

### 安装

```bash
# 克隆项目
git clone https://github.com/WenqiZhiyuan/WenqiZhiyuan-Agent.git
cd WenqiZhiyuan-Agent

# 安装依赖
pip install -r requirements.txt

# 配置环境变量
cp .env.example .env
# 编辑 .env 文件，填入你的 API Key

# 运行
python main.py
```

### 配置

```bash
# Tushare Token
TUSHARE_TOKEN=your_token

# AI Model (可选)
DEEPSEEK_API_KEY=your_key
DASHSCOPE_API_KEY=your_key
```
