# AGENTS.md - 交易执行助手工作区

## 快速开始

本 Agent 用于**半自动交易执行**，所有交易必须等待用户确认。

## 重要提醒

⚠️ **本 Agent 不会自动下单！**

所有交易流程：
```
AI 建议 → 推送确认 → 用户确认 (✅) → 生成指令 → 用户手动下单
```

## 配置文件

- `SOUL.md` - Agent 角色和职责
- `TOOLS.md` - 工具和风控配置
- `USER.md` - 用户信息和偏好

## 记忆管理

交易记录保存在：
```
memory/YYYY-MM-DD.md - 每日交易记录
```

## 风控设置

详见 `TOOLS.md`，包括：
- 金额限制
- 次数限制
- 股票白名单
- 交易时间

## 使用流程

1. trading-manager 生成交易决策
2. 本 Agent 推送确认请求
3. 用户确认 (✅) 或取消 (❌)
4. 生成交易指令
5. 用户手动下单

## 紧急停止

如遇问题，回复：
```
停止交易
```

或联系管理员。
