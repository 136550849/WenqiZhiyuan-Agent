## 🔄 标准工作流程

### 流程 1: 日常分析流程

```
┌─────────┐    ┌─────────┐    ┌─────────┐    ┌─────────┐
│  触发   │ →  │ 任务分配 │ →  │ 分析执行 │ →  │ 整合输出 │
└─────────┘    └─────────┘    └─────────┘    └─────────┘
   │              │              │              │
   │              │              │              │
   ▼              ▼              ▼              ▼
盘前/盘中/    trading-      各分析师       trading-
盘后触发      manager       独立分析       manager
            分配任务                      整合观点
                                          生成计划
```

#### 详细步骤

**步骤 1: 触发**
- **盘前** (9:00-9:25): 生成今日关注股票和交易计划
- **盘中** (9:30-15:00): 监控持仓，提示异常
- **盘后** (15:00 后): 生成复盘报告

**步骤 2: 任务分配**
- trading-manager 确定分析目标 (股票/行业)
- 分配任务给相关分析师
- 设定完成时限

**步骤 3: 分析执行**
- 各分析师独立执行分析
- 输出结构化分析报告
- 标注置信度和关键假设

**步骤 4: 整合输出**
- trading-manager 收集各方观点
- 综合分析矩阵评分
- 生成交易计划
- 传递给下游执行

## 📋 分析师职责矩阵

| 分析师 | 职责 | 输出 | 关键指标 |
|--------|------|------|----------|
| **market-analyst** 📊 | 技术面分析 | 支撑/阻力、趋势判断 | 准确率、及时率 |
| **fundamental-analyst** 💰 | 基本面分析 | 估值、财务健康度 | PE/PB、ROE、增长率 |
| **technical-analyst** 📈 | 深度技术分析 | 指标分析、形态识别 | MACD、RSI、MA |
| **news-analyst** 📰 | 消息面分析 | 重大事件、政策影响 | 事件影响评估 |
| **sentiment-analyst** 😊 | 情绪分析 | 市场情绪、资金流向 | 情绪指标、净流入 |
| **risk-analyst** ⚠️ | 风险评估 | 风险等级、风险收益比 | 最大回撤、波动率 |
| **research-lead** 👨‍🏫 | 研究统筹 | 研究计划、质量把控 | 研究深度、实用性 |
| **trader** 💹 | 交易执行 | 执行报告、操作日志 | 执行准确率、滑点 |
| **portfolio-manager** 💼 | 组合管理 | 仓位配置、再平衡建议 | 组合收益、夏普比率 |
| **post-processor** 📄 | 报告生成 | 正式报告、归档 | 报告质量、及时性 |
