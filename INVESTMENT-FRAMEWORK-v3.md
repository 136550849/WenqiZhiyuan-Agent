## 📖 第一部分：价值投资框架

### 1.1 格雷厄姆价值投资体系

#### 核心概念

**内在价值 (Intrinsic Value)**
- 企业未来现金流的折现值
- 不等同于市场价格
- 需要独立估算

**安全边际 (Margin of Safety)**
```
安全边际 = (内在价值 - 市场价格) / 内在价值

安全边际 > 30%: 高安全边际，可重仓
安全边际 10-30%: 中等安全边际，可轻仓
安全边际 < 10%: 低安全边际，观望
安全边际 < 0%: 无安全边际，回避/卖出
```

**市场先生 (Mr. Market)**
- 市场是为你服务的，不是指导你的
- 市场情绪波动创造买卖机会
- 利用市场，不被市场利用

#### 格雷厄姆选股标准

| 标准 | 要求 | 说明 |
|------|------|------|
| 规模 | 大型公司 | 避免小公司风险 |
| 财务实力 | 流动资产>流动负债 2 倍 | 偿债能力强 |
| 盈利稳定 | 近 10 年连续盈利 | 经营稳定 |
| 股息记录 | 近 20 年连续分红 | 股东回报 |
| 盈利增长 | 近 10 年 EPS 增长≥33% | 成长性 |
| 股价/净资产 | P/B < 1.5 | 估值合理 |
| 股价/盈利 | P/E < 15 | 估值合理 |

### 1.2 巴菲特投资体系

#### 四大原则

**1. 能力圈原则**
- 只投资自己懂的公司
- 能力圈大小不重要，边界清晰最重要
- 持续学习，扩大能力圈

**2. 护城河原则**
- 品牌优势 (可口可乐)
- 转换成本 (微软 Office)
- 网络效应 (微信、Facebook)
- 成本优势 (沃尔玛、亚马逊)
- 特许经营权 (贵州茅台)

**3. 管理层原则**
- 管理层是否理性
- 管理层是否坦诚
- 管理层是否抵抗惯性思维
- 管理层是否以股东利益为导向

**4. 财务原则**
- 关注 ROE (净资产收益率)
- 关注自由现金流
- 关注毛利率和净利率
- 关注负债率

#### 巴菲特选股标准

| 指标 | 标准 | 说明 |

| 净利率 | >20% | 盈利能力强 |
| 负债率 | <50% | 财务稳健 |
| 自由现金流 | 持续为正 | 造血能力强 |
| 营收增长 | >10% | 成长性 |

## 📊 第二部分：经济学基础

### 2.1 宏观经济学核心概念

#### 经济周期

**美林时钟**
```
          高增长
            │
    复苏    │    过热
   (股票)   │   (大宗商品)
            │
低通胀 ─────┼───── 高通胀
            │
    衰退    │    滞胀
   (债券)   │   (现金)
            │
          低增长
```

**周期阶段特征**

| 阶段 | GDP | CPI | 政策 | 最佳资产 |
|------|-----|-----|------|----------|
| 复苏 | ↑ | ↓ | 宽松 | 股票 |
| 过热 | ↑ | ↑ | 收紧 | 大宗商品 |
| 滞胀 | ↓ | ↑ | 收紧 | 现金 |
| 衰退 | ↓ | ↓ | 宽松 | 债券 |

#### 关键经济指标

**领先指标 (预测经济)**
- PMI (采购经理指数) >50 扩张
- 新订单指数
- 股票指数
- 货币供应量 M2

**同步指标 (反映当前)**
- GDP 增长率
- 工业增加值
- 社会消费品零售总额
- 就业数据

**滞后指标 (确认趋势)**
- CPI (消费者物价指数)
- 失业率
- 企业库存
- 贷款利率

### 2.2 微观经济学核心概念

#### 供需理论

```
价格
  │     供给曲线 S
  │    /
P*│───● 均衡点
  │  / \
  │ /   需求曲线 D
  │/
  └─────────── 数量
```

**应用**
- 供不应求 → 价格上涨 → 利好相关股票
- 供过于求 → 价格下跌 → 利空相关股票
- 寻找供需拐点

#### 竞争优势

**五种竞争力量 (波特五力)**
1. **现有竞争者**: 行业竞争激烈程度
2. **潜在进入者**: 进入壁垒高低
3. **替代品**: 替代威胁大小
4. **供应商议价能力**: 上游集中度
5. **客户议价能力**: 下游集中度

**护城河来源**
- 无形资产 (品牌、专利、牌照)
- 转换成本
- 网络效应
- 成本优势
- 有效规模

### 2.3 行为经济学核心概念

#### 认知偏差

| 偏差 | 描述 | 投资影响 |
|------|------|----------|
| **过度自信** | 高估自己的能力 | 频繁交易、集中持仓 |
| **确认偏误** | 只寻找支持自己观点的信息 | 忽视风险信号 |
| **损失厌恶** | 损失带来的痛苦大于收益的快乐 | 死扛亏损、过早止盈 |
| **锚定效应** | 过度依赖初始信息 | 被买入价锚定 |
| **从众效应** | 跟随大众行为 | 追涨杀跌 |
| **可得性偏差** | 依据容易想起的信息判断 | 过度关注近期事件 |
| **代表性偏差** | 依据刻板印象判断 | 以偏概全 |

#### 应对策略

1. **建立投资清单**: 避免情绪化决策
2. **逆向思考**: 第二层次思维
3. **记录决策过程**: 事后复盘
4. **设定规则**: 严格执行止损/止盈
5. **多元化**: 避免单一判断错误

## 🎯 第三部分：资产配置模型

### 3.1 战略资产配置

#### 经典模型对比

**1. 60/40 组合**
```
股票：60%
债券：40%

特点：简单、经典、长期有效
适合：稳健型投资者
```

**2. 永久组合 (Harry Browne)**
```
股票：25% (经济增长)
债券：25% (通货紧缩)
黄金：25% (通货膨胀)
现金：25% (经济衰退)

特点：全天候、低波动
适合：保守型投资者
```

**3. 全天候策略 (Ray Dalio)**
```
股票：30% (经济增长)
长期国债：40% (通货紧缩)
中期国债：15% (平衡)
黄金：7.5% (通货膨胀)
大宗商品：7.5% (通货膨胀)

特点：风险平价、低相关性
适合：机构投资者
```

**4. 耶鲁模式 (David Swensen)**
```
美国股票：30%
外国股票：15%
房地产：20%
长期国债：15%
中期国债：5%
通胀保值债券：15%

特点：多元化、另类投资
适合：长期资金
```

### 3.2 战术资产配置

#### 基于估值的配置

**股票仓位公式**
```
股票仓位 = 基础仓位 × (1 + 估值调整)

估值调整 = (历史平均 PE - 当前 PE) / 历史平均 PE

例：
基础仓位 = 60%
历史平均 PE = 15
当前 PE = 12

估值调整 = (15 - 12) / 15 = 20%
股票仓位 = 60% × (1 + 20%) = 72%
```

#### 基于趋势的配置

**趋势判断指标**
- 200 日均线：价格在上方为上升趋势
- MACD: 金叉为上升趋势
- 成交量：放大确认趋势

**配置规则**
```
上升趋势：股票仓位 70-90%
震荡趋势：股票仓位 40-60%
下降趋势：股票仓位 10-30%
```

### 3.3 再平衡策略

#### 定期再平衡

**频率**
- 月度：过于频繁，交易成本高
- 季度：推荐，平衡成本和效果
- 年度：可能偏离过大

**方法**
```
原始配置：股票 60%, 债券 40%
一年后：股票 70%, 债券 30% (股票上涨)

再平衡操作：
卖出 10% 股票，买入 10% 债券
恢复至 60/40 配置
```

#### 阈值再平衡

**规则**
```
当某类资产偏离目标配置±5% 时再平衡

例：目标股票 60%
- 股票>65%: 卖出至 60%
- 股票<55%: 买入至 60%
```

**优点**
- 避免频繁交易
- 抓住大波动机会
- 降低交易成本

## ⚠️ 第四部分：风险管理框架

### 4.1 风险类型

| 风险类型 | 描述 | 应对方法 |
|----------|------|----------|
| **市场风险** | 系统性风险，无法分散 | 降低仓位、对冲 |
| **公司风险** | 个体公司经营风险 | 分散投资、深入研究 |
| **流动性风险** | 无法及时变现 | 保持现金、选择高流动性资产 |
| **汇率风险** | 汇率波动影响 | 对冲、本币资产为主 |
| **通胀风险** | 购买力下降 | 配置抗通胀资产 (股票、房产、黄金) |
| **利率风险** | 利率变化影响债券价格 | 缩短久期、浮动利率债券 |
| **信用风险** | 债券违约 | 选择高评级债券、分散 |
| **政策风险** | 政策变化影响 | 关注政策、及时调整 |

### 4.2 风险度量

#### 波动率 (标准差)

```
波动率 = 收益率的标准差

解读:
- 波动率<10%: 低风险
- 波动率 10-20%: 中等风险
- 波动率>20%: 高风险
```

#### 最大回撤

```
最大回撤 = (历史最高值 - 历史最低值) / 历史最高值

解读:
- 最大回撤<10%: 低风险
- 最大回撤 10-30%: 中等风险
- 最大回撤>30%: 高风险
```

#### VaR (风险价值)

```
VaR(95%, 1 天) = 100 万元

解读:
在 95% 的置信水平下，1 天内最大损失不超过 100 万元
```

### 4.3 风险控制

#### 仓位控制

**单只股票上限**
- 保守型：5%
- 稳健型：10%
- 激进型：20%

**单行业上限**
- 保守型：20%
- 稳健型：30%
- 激进型：40%

**总仓位控制**
- 牛市：70-90%
- 震荡市：40-60%
- 熊市：10-30%

#### 止损策略

| 止损类型 | 方法 | 适用场景 |
|----------|------|----------|
| **固定比例止损** | -5% / -8% / -10% | 短线交易、波动大的标的 |
| **技术位止损** | 跌破关键支撑位 | 趋势交易、技术分析 |
| **时间止损** | X 日内无表现则离场 | 事件驱动、预期落空 |
| **基本面止损** | 基本面恶化 | 价值投资、长期持有 |
| **移动止损** | 跟踪最高点回撤 X% | 趋势行情、保护利润 |

#### 分散化

**资产分散**
- 股票、债券、现金、黄金、大宗商品
- 相关性低的资产组合

**地域分散**
- A 股、港股、美股
- 发达市场、新兴市场

**行业分散**
- 消费、科技、金融、医疗、制造
- 避免单一行业集中

**时间分散**
- 分批建仓
- 定投策略

## 📈 第五部分：估值方法

### 5.1 相对估值法

#### PE (市盈率)

```
PE = 股价 / 每股收益 (EPS)

适用：盈利稳定的公司
合理区间：10-25 倍 (视行业而定)

注意:
- 周期股 PE 低时可能是高点
- 成长股 PE 高可能合理
- 对比历史 PE 和同行 PE
```

#### PB (市净率)

```
PB = 股价 / 每股净资产

适用：重资产公司 (银行、保险、制造)
合理区间：1-5 倍 (视行业而定)

注意:
- PB<1 可能被低估
- 轻资产公司 PB 参考意义小
```

#### PEG (市盈率相对盈利增长比率)

```
PEG = PE / 盈利增长率

适用：成长型公司
合理区间：
- PEG<1: 低估
- PEG 1-2: 合理
- PEG>2: 高估
```

#### PS (市销率)

```
PS = 市值 / 营业收入

适用：未盈利的成长型公司 (互联网、生物科技)
合理区间：视行业而定

注意:
- 关注营收增长率
- 关注毛利率变化
```

### 5.2 绝对估值法

#### DCF (现金流折现)

```
内在价值 = Σ (未来自由现金流 / (1 + 折现率)^n)

步骤:
1. 预测未来 5-10 年自由现金流
2. 选择折现率 (通常 8-12%)
3. 计算终值
4. 折现求和

优点: 理论基础扎实
缺点: 对假设敏感，预测困难

适用：现金流稳定的公司
```

#### DDM (股利折现模型)

```
内在价值 = 下一年股利 / (折现率 - 股利增长率)

适用：稳定分红公司 (银行、公用事业)

注意:
- 要求股利稳定增长
- 折现率>股利增长率
```

### 5.3 估值比较

#### 历史估值比较

```
当前 PE vs 历史 PE 分位

例:
当前 PE = 15
历史 PE 范围：10-30
历史 PE 中位数：20

当前 PE 分位 = (15-10) / (30-10) = 25%

解读: 当前估值低于历史 75% 的时间，低估
```

#### 同行估值比较

```
公司 PE vs 行业平均 PE

例:
公司 PE = 20
行业平均 PE = 25

解读: 公司估值低于行业平均，可能低估
注意: 考虑公司质量、成长性差异
```

## 🎓 第六部分：投资决策流程

### 6.1 选股流程

```
1. 初筛 (量化筛选)
   │
   ├─ 市值 > 100 亿
   ├─ ROE > 15%
   ├─ 营收增长 > 10%
   ├─ 净利润增长 > 10%
   ├─ 负债率 < 50%
   └─ 毛利率 > 30%
   │
   ▼
2. 深入研究 (定性分析)
   │
   ├─ 商业模式分析
   ├─ 护城河分析
   ├─ 管理层分析
   ├─ 行业前景分析
   └─ 竞争格局分析
   │
   ▼
3. 财务分析 (定量分析)
   │
   ├─ 三大报表分析
   ├─ 财务比率分析
   ├─ 现金流分析
   └─ 盈利质量分析
   │
   ▼
4. 估值分析
   │
   ├─ 相对估值 (PE/PB/PEG)
   ├─ 绝对估值 (DCF)
   ├─ 历史估值比较
   └─ 同行估值比较
   │
   ▼
5. 安全边际计算
   │
   ├─ 内在价值估算
   ├─ 安全边际计算
   └─ 买入价格确定
   │
   ▼
6. 决策
   │
   ├─ 安全边际>30%: 重仓买入
   ├─ 安全边际 10-30%: 轻仓买入
   ├─ 安全边际<10%: 观望
   └─ 安全边际<0%: 回避/卖出
```

### 6.2 仓位决策流程

```
1. 市场状态判断
   │
   ├─ 宏观环境分析
   ├─ 市场情绪分析
   └─ 估值水平分析
   │
   ▼
2. 确定基础仓位
   │
   ├─ 牛市：70-90%
   ├─ 震荡市：40-60%
   └─ 熊市：10-30%
   │
   ▼
3. 个股分析
   │
   ├─ 置信度评估 (高/中/低)
   ├─ 风险评估 (高/中/低)
   └─ 安全边际评估
   │
   ▼
4. 计算建议仓位
   │
   建议仓位 = 基础仓位 × 置信度系数 × 风险调整 × 安全边际
   │
   ▼
5. 应用仓位上限
   │
   ├─ 单只股票上限
   ├─ 单行业上限
   └─ 总仓位上限
   │
   ▼
6. 制定建仓计划
   │
   ├─ 分批建仓 (3 批)
   ├─ 价格区间确定
   └─ 触发条件确定
```

### 6.3 卖出决策流程

```
卖出信号:

1. 估值过高
   - PE/PB 远高于历史均值
   - 安全边际消失或为负
   - 市值远超内在价值

2. 基本面恶化
   - 营收/利润增速大幅下滑
   - 毛利率/净利率下降
   - 负债率大幅上升
   - 现金流恶化

3. 护城河受损
   - 竞争优势丧失
   - 市场份额下降
   - 品牌力下降

4. 发现更好的机会
   - 有更高安全边际的标的
   - 资金有限需要调仓

5. 止损触发
   - 达到预设止损位
   - 基本面判断错误

卖出策略:
- 倒金字塔减仓 (分 3-4 批)
- 移动止盈 (保护利润)
- 目标价止盈 (达到估值目标)
```
