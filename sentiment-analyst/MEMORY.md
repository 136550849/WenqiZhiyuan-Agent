# MEMORY.md - 情绪分析师的长期记忆

## 🧠 核心认知

### 情绪分析的本质
- 情绪是市场波动的重要驱动因素，而非噪音
- 情绪分析的目标不是预测，而是识别极端状态
- 情绪指标需要多维度验证，单一指标不可靠

### 行为金融学基础
- **损失厌恶**：损失带来的痛苦约是同等收益快乐的 2.25 倍
- **过度自信**：投资者普遍高估自己的判断能力
- **从众行为**：群体压力导致非理性决策
- **锚定效应**：过度依赖初始信息
- **确认偏误**：只关注支持自己观点的信息

### 情绪周期规律
- 绝望期（极度悲观）→ 底部区域，关注反弹机会
- 怀疑期（谨慎悲观）→ 反弹初期
- 乐观期（逐步看好）→ 上涨中期
- 狂热期（极度乐观）→ 顶部区域，警惕回调风险

## 📊 关键指标阈值

| 指标 | 极度悲观 | 悲观 | 中性 | 乐观 | 极度乐观 |
|------|---------|------|------|------|---------|
| 情绪得分 | <-70 | -70~-30 | -30~30 | 30~70 | >70 |
| 贪婪恐惧指数 | <10 | 10~30 | 30~70 | 70~90 | >90 |
| 看涨看跌比 | <0.5 | 0.5~1 | 1~1.5 | 1.5~2 | >2 |
| 讨论热度百分位 | <10% | 10~30% | 30~70% | 70~90% | >90% |

## 🎯 逆向投资原则

### 极度乐观时的行动
1. 警惕"这次不一样"的论调
2. 检查基本面是否支撑高估值
3. 考虑减仓或提高止损位
4. 不追高，等待情绪回落

### 极度悲观时的行动
1. 关注"世界末日"论调是否过度
2. 检查是否有错杀机会
3. 分批建仓，设置止损
4. 保持耐心，底部可能持续较长时间

## 🤝 协作关系

- **trading-manager**：接收分析任务，传递分析结果
- **news-analyst**：交叉验证消息面
- **risk-analyst**：评估情绪风险
- **technical-analyst**：验证情绪与技术面背离

## ⚠️ 分析原则

1. 数据驱动，不主观臆测
2. 多维度验证，综合判断
3. 历史对比，判断极端程度
4. 逆向思考，极端时保持警惕
5. 及时更新，情绪变化快
6. 行为视角，用行为金融学解读
7. 周期定位，明确情绪在周期中的位置

## 🔍 关键洞察

1. **情绪分析必须结合行为金融学原理** - 用理论解释现象
2. **单一情绪指标不可靠** - 需要多维度验证
3. **极端情绪是逆向投资信号** - 但需要基本面确认
4. **情绪周期与基本面周期可能不同步** - 产生背离机会
5. **情绪分析的目标不是预测** - 而是识别极端状态
