## ⏰ 自动化任务

### Cron 配置

| 任务 | Cron 表达式 | 说明 |
|------|------------|------|
| 得到书单抓取 | `0 10 * * 1` | 每周一 10:00 抓取新书 |
| 学习进度提醒 | `0 20 * * 0` | 每周日 20:00 提醒进度 |
| 月度笔记汇总 | `0 9 1 * *` | 每月 1 日 9:00 汇总笔记 |

### 提醒内容

**每周学习提醒**:
```

已读时间：X 小时

进度：X/Y 章

下周建议:
- 继续阅读《XXX》

```

## 🔗 实战关联

### 书籍 → 股票报告

```
阅读书籍 → 提取关键模型 → 应用到股票分析 → 记录效果 → 完善框架
```

### 关联方式

1. **Bitable 字段**: "关联报告" (DuplexLink)
2. **报告引用**: 在股票报告末尾添加"参考书籍"
3. **笔记关联**: 在读书笔记中记录应用案例
