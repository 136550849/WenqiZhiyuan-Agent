## [1.0.0] - 2026-03-25

### 🎉 首次正式发布

#### 新增

**AI 记忆索引系统**
- 三层记忆结构（长期/短期/工作记忆）
- 记忆分类体系（6 大分类）
- 记忆检索机制（语义搜索）
- 记忆管理流程（每日/每周/每月）
- 质量评估标准（5 个维度）

**电子书库管理模块**
- `scanner.py` - Calibre 书库扫描器
  - 支持 6 种格式（epub/azw3/pdf/mobi/txt/fb2）
  - 流式扫描（节省内存）
  - 并行扫描（多线程/多进程）
  - 智能缓存（24 小时有效期）
- `metadata.py` - 元数据提取器
  - EPUB 完整元数据提取
  - AZW3 元数据（需要 Calibre）
  - PDF 元数据（PyMuPDF）
  - 批量提取支持
- `quality.py` - 质量评估器
  - A/B/C/D 四级评估体系
  - 多维度评分（大小/元数据/格式）
  - 批量评估与统计
  - Markdown 报告生成
- `dedup.py` - 重复检测器
  - 智能检测（ISBN/书名 + 作者）
  - 严格检测（完全匹配）
  - 宽松检测（书名模糊匹配）
  - 文件哈希检测
  - 多策略去重
- `organizer.py` - 分类整理器
  - 领域分类（10 个领域）
  - 难度分级（入门/进阶/高级/研究）
  - 优先级分类（P0/P1/P2/P3）
  - Bitable 导出支持
- `performance.py` - 性能优化模块
  - 流式扫描（节省内存 5 倍）
  - 并行扫描（提速 4-6 倍）
  - 智能缓存（提速 240 倍）
  - 分块处理（内存优化）
  - 懒加载（按需读取）

**传承机制文档**
- `inheritance_sop.md` - 传承机制标准作业程序
  - 新人入门流程（1 天/1 周/1 月/3 月）
  - 知识更新机制（每日/每周/每月/每季）
  - 经验传承方法（师徒制/案例教学/文档沉淀）
  - 风险评估与应对
  - 效果评估指标

**文档与教程**
- `README.md` - 完整设计文档（410 行）
- `TUTORIAL.md` - 使用教程（420 行）
  - 快速开始（6 步流程）
  - 高级用法
  - 常见问题解答
  - 最佳实践
- `PERFORMANCE_GUIDE.md` - 性能优化指南（320 行）
  - 性能基准测试
  - 优化技术详解
  - 实际案例
  - 配置建议
- `RELEASE_v1.0.0.md` - 发布说明（410 行）

**测试与示例**
- `test_ebook_library.py` - 单元测试脚本
  - 5 个模块测试
  - 自动化测试报告
- `examples/01_quick_start.py` - 快速开始示例
- `examples/02_batch_process.py` - 批量处理示例（多线程）

#### 性能

- 扫描速度：120 秒 → 25 秒（**4.8 倍**）
- 元数据提取：300 秒 → 60 秒（**5.0 倍**）
- 内存占用：2.5GB → 500MB（**5.0 倍**）
- 重复扫描：120 秒 → 0.5 秒（**240 倍**）
- 支持规模：最高 10 万 + 书籍

#### 技术栈

- Python 3.8+
- ebooklib >= 0.18
- beautifulsoup4 >= 4.12
- PyMuPDF >= 1.23

## 版本说明

### 语义化版本

- **主版本号 (Major)**: 不兼容的 API 变更
- **次版本号 (Minor)**: 向下兼容的功能性新增
- **修订号 (Patch)**: 向下兼容的问题修正

### 发布周期

- **v1.x.x**: 每月发布一个次版本
- **v1.0.x**: 每周发布一个修订版
- **v2.0.0**: 重大更新（季度）
