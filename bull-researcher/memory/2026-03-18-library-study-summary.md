2. **投资心理学**: 市场先生、逆向思维、情绪控制
3. **资产配置**: 分散投资、股债平衡、ETF 配置
4. **财务基础**: 三张报表、财务比率、现金流分析

### 二、行业分析类 (02-行业分析)

**代表书籍**:
- 行业研究方法论相关
- 产业链分析框架
- 竞争格局分析
- 行业生命周期理论

**核心知识点**:
1. **行业分析框架**: 市场规模、增长率、竞争格局
2. **产业链分析**: 上下游关系、议价能力、价值链分布
3. **波特五力模型**: 供应商/买方议价力、新进入者威胁、替代品威胁、现有竞争
4. **行业生命周期**: 导入期、成长期、成熟期、衰退期

---

### 三、财务分析类 (03-财务分析)

**代表书籍**:
- 财务报表分析相关
- 估值建模方法
- DCF 现金流折现
- 相对估值法 (PE/PB/PS/PEG)

**核心知识点**:
1. **三张报表分析**:
   - 资产负债表：资产结构、负债水平、所有者权益
   - 利润表：收入、成本、利润、增长率
   - 现金流量表：经营/投资/筹资活动现金流

2. **财务比率分析**:
   - 盈利能力：ROE、ROA、毛利率、净利率
   - 偿债能力：资产负债率、流动比率、速动比率
   - 营运能力：周转率、周转天数
   - 成长能力：收入增速、利润增速

3. **估值方法**:
   - DCF 模型：自由现金流折现
   - 相对估值：PE、PB、PS、PEG
   - 分部估值：各业务板块分别估值
   - 清算价值：最保守估值
