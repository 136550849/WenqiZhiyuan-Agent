- 三层记忆结构（长期/短期/工作记忆）
- 语义搜索支持

### 🔧 技术栈

- **Python**: 3.8+
- **核心依赖**:
  - ebooklib >= 0.18
  - beautifulsoup4 >= 4.12
  - PyMuPDF >= 1.23
- **可选依赖**:
  - Calibre（用于 azw3 元数据）
  - 飞书 Bitable（用于数据导出）

### 📥 安装指南

```bash
# 1. 克隆项目
git clone https://github.com/136550849/WenqiZhiyuan-Agent.git
cd WenqiZhiyuan-Agent

# 2. 安装依赖
cd modules/ebook_library
pip install -r requirements.txt

# 3. 运行测试
python test_ebook_library.py

# 4. 快速开始
python examples/01_quick_start.py
```

### 🐛 已知问题

1. **azw3 元数据提取**需要安装 Calibre
   - 临时方案：使用 ebook-meta 命令行工具
   - 计划：v1.1 添加纯 Python 实现

2. **超大型书库**（>50 万本）内存占用较高
   - 临时方案：减小 chunk_size 到 200
   - 计划：v1.2 添加数据库支持

3. **PDF 元数据**提取不完整
   - 临时方案：手动补充元数据
   - 计划：v1.1 优化 PDF 解析
