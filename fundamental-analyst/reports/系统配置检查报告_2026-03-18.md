## 🌐 Web 搜索配置

### 3.1 Brave Search API

| 项目 | 状态 | 说明 |
|------|------|------|
| API Key | ❌ **未配置** | 需要配置 |
| 配置路径 | `web.brave.apiKey` | - |

**影响：**
- ❌ web_search 工具无法使用
- ❌ 无法搜索实时新闻和数据
- ❌ 无法获取最新财报信息

**解决方案：**
```bash
openclaw configure --section web
# 或手动编辑配置文件添加:
"web": {
  "brave": {
    "apiKey": "YOUR_BRAVE_API_KEY"
  }
}
```

**获取 API Key：**
1. 访问 https://brave.com/search/api/
2. 注册账号并创建 API Key
3. 免费额度：每月 2000 次搜索
