### 步骤 4：财务健康度分析

#### 资产负债率
```
公式：资产负债率 = 总负债 / 总资产 × 100%

判断标准：
- <40%：财务稳健
- 40%-60%：合理水平
- >60%：财务风险较高
- >70%：财务风险高
```

#### 流动比率 & 速动比率
```
流动比率 = 流动资产 / 流动负债
- >2：短期偿债能力强
- 1-2：短期偿债能力一般
- <1：短期偿债能力弱

速动比率 = (流动资产 - 存货) / 流动负债
- >1：短期偿债能力强
- 0.5-1：短期偿债能力一般
- <0.5：短期偿债能力弱
```

#### 有息负债率
```
有息负债率 = 有息负债 / 总资产 × 100%

有息负债包括：
- 短期借款
- 长期借款
- 应付债券

判断标准：
- <20%：财务安全
- 20%-40%：合理水平
- >40%：财务压力较大
```

#### 现金流状况
```
经营现金流分析：
- 经营现金流净额 > 净利润 → 盈利质量高
- 经营现金流净额 < 净利润 → 盈利质量低
- 经营现金流净额 < 0 → 经营困难

现金流充裕度：
- 货币资金 / 流动负债 > 1 → 资金充裕
- 货币资金 / 流动负债 < 0.5 → 资金紧张
```

### 步骤 5：盈利能力分析

#### ROE（净资产收益率）
```
公式：ROE = 净利润 / 净资产 × 100%

杜邦分析：
ROE = 净利率 × 周转率 × 杠杆率
    = (净利润/营收) × (营收/资产) × (资产/净资产)

判断标准：
- ROE > 20%：盈利能力极强
- ROE 15%-20%：盈利能力强
- ROE 10%-15%：盈利能力中等
- ROE < 10%：盈利能力弱

持续性分析：
- 连续 5 年 ROE > 15% → 优质公司
- ROE 波动大 → 盈利不稳定
```

#### 毛利率
```
公式：毛利率 = (营业收入 - 营业成本) / 营业收入 × 100%

判断标准：
- 毛利率 > 50%：高毛利，竞争力强
- 毛利率 30%-50%：中等毛利
- 毛利率 < 30%：低毛利，竞争激烈

趋势分析：
- 毛利率上升 → 竞争力增强
- 毛利率下降 → 竞争压力增大
```

#### 净利率
```
公式：净利率 = 净利润 / 营业收入 × 100%

判断标准：
- 净利率 > 20%：盈利能力强
- 净利率 10%-20%：盈利能力中等
- 净利率 < 10%：盈利能力弱

对比分析：
- 净利率 vs 行业平均
- 净利率 vs 历史水平
```

#### ROA（总资产收益率）
```
公式：ROA = 净利润 / 总资产 × 100%

判断标准：
- ROA > 10%：资产利用效率高
- ROA 5%-10%：资产利用效率中等
- ROA < 5%：资产利用效率低
```

### 步骤 6：成长能力分析

#### 营收增长率
```
公式：营收增长率 = (本期营收 - 上期营收) / 上期营收 × 100%

判断标准：
- 增长率 > 30%：高速增长
- 增长率 15%-30%：中速增长
- 增长率 5%-15%：低速增长
- 增长率 < 5%：增长停滞
- 增长率 < 0%：负增长

持续性分析：
- 连续 3 年增长 > 20% → 高成长公司
- 增长波动大 → 业务不稳定
```

#### 净利润增长率
```
公式：净利润增长率 = (本期净利润 - 上期净利润) / 上期净利润 × 100%

判断标准：
- 增长率 > 30%：利润高速增长
- 增长率 15%-30%：利润中速增长
- 增长率 5%-15%：利润低速增长
- 增长率 < 5%：利润增长停滞
- 增长率 < 0%：利润下滑

质量分析：
- 利润增长 > 营收增长 → 盈利能力提升
- 利润增长 < 营收增长 → 盈利能力下降
```

#### 可持续增长率
```
公式：可持续增长率 = ROE × (1 - 分红率)

含义：在不增加财务杠杆的情况下，公司可实现的最大增长率

应用：
- 实际增长率 > 可持续增长率 → 可能面临资金压力
- 实际增长率 < 可持续增长率 → 增长潜力未充分释放
```

### 步骤 7：估值分析

#### PE（市盈率）
```
公式：PE = 股价 / 每股收益（或市值 / 净利润）

类型：
- PE(TTM)：滚动市盈率（最近 12 个月）
- PE(静)：静态市盈率（上一年度）
- PE(动)：动态市盈率（预测）

判断标准（因行业而异）：
- PE < 10：低估
- PE 10-20：合理
- PE 20-30：偏高
- PE > 30：高估

对比分析：
- PE vs 行业平均
- PE vs 历史 PE 区间
- PE vs 增长率（PEG）
```

#### PB（市净率）
```
公式：PB = 股价 / 每股净资产（或市值 / 净资产）

判断标准：
- PB < 1：破净，可能低估
- PB 1-3：合理
- PB 3-5：偏高
- PB > 5：高估

适用性：
- 适合重资产行业（银行、钢铁、地产）
- 不适合轻资产行业（互联网、软件）
```

#### PEG（市盈率相对盈利增长比率）
```
公式：PEG = PE / 净利润增长率

判断标准：
- PEG < 1：低估（增长快于估值）
- PEG 1-1.5：合理
- PEG > 1.5：高估（估值快于增长）

优势：
- 考虑了增长因素
- 适合成长股估值
```

#### DCF（现金流折现）
```
基本公式：
企业价值 = ∑(自由现金流 / (1+WACC)^t) + 终值

关键假设：
- 自由现金流预测（5-10 年）
- 折现率（WACC）
- 永续增长率

适用性：
- 适合现金流稳定的公司
- 不适合现金流波动大的公司
```

### 步骤 8：同业对比

#### 对比维度
```
选择可比公司：
- 同一行业
- 规模相近
- 业务相似

对比指标：
- 估值指标：PE、PB、PS、EV/EBITDA
- 盈利能力：ROE、毛利率、净利率
- 成长能力：营收增长、利润增长
- 财务健康：资产负债率、流动比率
```

#### 相对估值
```
方法：
1. 计算行业平均估值
2. 对比目标公司估值
3. 判断高估/低估

示例：
- 行业平均 PE：20 倍
- 目标公司 PE：15 倍
- 结论：相对低估 25%
```

### 步骤 9：护城河分析

#### 护城河类型
```
1. 无形资产
   - 品牌优势（如茅台）
   - 专利保护（如医药）
   - 牌照壁垒（如金融）

2. 成本优势
   - 规模效应
   - 资源优势
   - 工艺优势

3. 网络效应
   - 用户越多价值越大
   - 如社交平台、电商平台

4. 转换成本
   - 客户更换供应商成本高
   - 如企业软件、医疗器械

5. 规模优势
   - 市场份额领先
   - 行业龙头地位
```
