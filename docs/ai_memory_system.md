## 🔍 记忆检索机制

### 语义搜索

```python
# 记忆检索接口
def search_memory(query, filters=None):
    """
    语义搜索记忆
    
    Args:
        query: 搜索关键词
        filters: 过滤条件 (分类、日期等)
    
    Returns:
        相关记忆片段列表
    """
    # 1. 全文搜索
    results = full_text_search(query)
    
    # 2. 语义相似度计算
    results = rank_by_semantic_similarity(results, query)
    
    # 3. 返回 Top N 结果
    return results[:10]
```
