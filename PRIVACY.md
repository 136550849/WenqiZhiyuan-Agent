## 📌 隐私保护承诺

**我们重视并保护您的隐私。本隐私政策说明我们如何收集、使用和保护您的信息。**

---

## 1. 信息收集 📊

### 1.1 本地配置信息

**项目可能在您的本地设备存储以下信息：**

| 信息类型 | 存储位置 | 用途 |
|----------|----------|------|
| API Token | 本地 .env 文件 | 调用第三方 API |
| 配置参数 | 本地配置文件 | 个性化设置 |
| 缓存数据 | 本地 cache 目录 | 提高性能 |
| 日志文件 | 本地 logs 目录 | 问题排查 |

### 1.2 不收集的信息

**我们不会收集以下信息：**

- ❌ 个人身份信息（姓名、电话、邮箱等）
- ❌ 财务信息（银行账户、交易密码等）
- ❌ 交易记录（持仓、买卖记录等）
- ❌ 设备信息（IP 地址、MAC 地址等）

### 1.3 数据存储

**所有数据仅存储在您的本地设备。**

- 数据不会上传到我们的服务器
- 不会与第三方共享
- 您随时可以删除本地数据

---

## 2. 信息使用 🛠️

### 2.1 使用目的

**收集的信息仅用于以下目的：**

- 提供项目功能（如调用 API）
- 优化性能（如缓存）
- 问题排查（如日志）
- 改进产品（如使用统计）

### 2.2 使用限制

**我们承诺：**

- ✅ 仅用于上述目的
- ✅ 不会出售或出租给第三方
- ✅ 不会用于营销目的
- ✅ 不会用于训练 AI 模型

**使用第三方 API 需要您自行提供密钥。**

- 请妥善保管 API 密钥

- 密钥泄露导致的损失由您自行承担

## 4. 数据安全 🔒

### 4.1 安全措施

**我们采取以下安全措施：**

- ✅ 敏感信息存储在本地
- ✅ .env 文件已加入 .gitignore
- ✅ 提供 .env.example 模板
- ✅ 文档中包含安全提示

### 4.2 您的责任

**您应自行保护本地数据安全：**

- 设置强密码保护设备
- 定期备份重要数据
- 不要分享敏感配置文件
- 及时更新软件和依赖

### 4.3 安全漏洞

**如发现安全漏洞，请及时报告。**

- 通过 GitHub Issues 报告
- 或通过邮件联系我们
- 我们会尽快修复问题

### 5.3 数据删除

**您可以随时删除本地数据：**

```bash
# 删除配置
rm .env

# 删除缓存
rm -rf cache/

# 删除日志
rm -rf logs/

# 删除所有本地数据
rm -rf .env cache/ logs/
```
