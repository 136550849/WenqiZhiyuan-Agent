## 🎯 学习收获与框架升级

### 一、价值投资核心原则（巴菲特体系）

1. **护城河理论**
   - 无形资产（品牌、专利、牌照）
   - 成本优势
   - 网络效应
   - 转换成本
   - 规模优势

2. **安全边际**
   - 价格 < 内在价值
   - 保守估计未来现金流
   - 预留犯错空间

3. **能力圈**
   - 只投资自己理解的行业
   - 持续扩展认知边界
   - 承认无知比假装知道更重要

4. **长期主义**
   - 持有期限：永远（如果可能）
   - 忽略短期波动
   - 关注企业长期竞争力

### 二、财务分析框架升级

#### 1. 三张表勾稽关系

**利润表 → 资产负债表**
- 净利润 → 未分配利润
- 折旧摊销 → 固定资产净值减少
- 应收账款增加 → 收入确认但未收款

**利润表 → 现金流量表**
- 净利润 → 经营活动现金流起点
- 折旧摊销 → 加回（非现金支出）
- 营运资本变动 → 调整项

**资产负债表 → 现金流量表**
- 固定资产增加 → 投资活动现金流流出
- 借款增加 → 筹资活动现金流流入
- 分红 → 筹资活动现金流流出

#### 2. ROE 杜邦分析深化

```
ROE = 净利率 × 总资产周转率 × 权益乘数

高 ROE 的三种模式：
1. 高净利率型（茅台、苹果）- 产品溢价能力强
2. 高周转型（沃尔玛、亚马逊）- 运营效率高
3. 高杠杆型（银行、地产）- 财务杠杆高（风险较高）

可持续 ROE 判断：
- ROE > 15% 持续 5 年以上 = 优秀
- 需分析 ROE 来源是否可持续
- 警惕杠杆驱动的高 ROE
```

#### 3. 财务操纵识别清单

**红色警报信号**：
- [ ] 应收账款增速 > 收入增速
- [ ] 存货增速 > 收入增速
- [ ] 经营性现金流持续 < 净利润
- [ ] 频繁变更会计政策
- [ ] 关联交易占比过高
- [ ] 毛利率显著高于同行且无合理解释
- [ ] 研发费用资本化比例异常
- [ ] 其他应收款/其他应付款异常增长

### 三、估值建模方法升级

#### 1. 相对估值法优化

| 估值指标 | 适用行业 | 注意事项 |
|---------|---------|---------|
| PE | 稳定盈利行业 | 警惕周期性、一次性损益 |
| PB | 重资产、金融 | 关注 ROE 水平 |
| PS | 高增长未盈利 | 需结合毛利率分析 |
| EV/EBITDA | 资本密集型 | 排除折旧政策影响 |
| PEG | 成长股 | 增长率预测准确性关键 |

#### 2. DCF 模型关键假设

**自由现金流计算**：
```
FCF = 息税前利润 (EBIT) × (1 - 税率) + 折旧摊销 - 资本开支 - 营运资本增加

或：
FCF = 经营活动现金流 - 资本开支
```

**关键假设敏感性**：
- 收入增长率：±1% 对估值影响约 10-15%
- 永续增长率：通常取 2-3%（不超过长期 GDP 增速）
- WACC：±0.5% 对估值影响约 5-10%

**WACC 计算**：
```
WACC = (E/V) × Re + (D/V) × Rd × (1 - Tc)

其中：
E = 股权市值
D = 债务市值
V = E + D
Re = 股权成本（CAPM: Rf + β × (Rm - Rf)）
Rd = 债务成本
Tc = 企业所得税率
```

### 四、研究流程优化

#### 标准研究流程（6 步法升级）

```
1. 信息收集（1-2 天）
   ├─ 年报/季报（3-5 年）
   ├─ 招股书/公告
   ├─ 券商研报（3-5 家）
   └─ 行业数据

2. 业务分析（1-2 天）
   ├─ 收入结构拆解
   ├─ 商业模式画布
   ├─ 护城河评估
   └─ 竞争格局分析

3. 财务分析（1-2 天）
   ├─ 三张表勾稽检查
   ├─ 杜邦分析
   ├─ 财务健康度评估
   └─ 操纵风险排查

4. 管理层评估（0.5 天）
   ├─ 背景调查
   ├─ 持股与激励
   ├─ 历史言行一致性
   └─ 资本配置记录

5. 估值建模（1-2 天）
   ├─ 相对估值（3-5 种方法）
   ├─ DCF 绝对估值
   ├─ 敏感性分析
   └─ 安全边际计算

6. 报告撰写（1 天）
   ├─ 执行摘要
   ├─ 核心逻辑
   ├─ 风险提示
   └─ 投资建议
```
