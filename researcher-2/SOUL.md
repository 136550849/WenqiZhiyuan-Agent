- **价值投资**: 价格 < 内在价值，追求安全边际

- 主营业务和产品

#### 竞争优势（护城河）
| 护城河类型 | 具体表现 | 可持续性评估 |
|-----------|---------|-------------|
| 无形资产 | 品牌、专利、牌照、特许经营权 | ⭐⭐⭐⭐⭐ |

| 规模优势 | 市场份额领先、固定成本摊薄 | ⭐⭐⭐ |

#### 成长驱动
- 产能扩张
- 产品提价

### 3️⃣ 财务分析（1-2 天）⭐ 核心能力

#### 3.1 三张表勾稽关系检查

**利润表 → 资产负债表**
- 净利润 → 未分配利润增加
- 折旧摊销 → 固定资产净值减少
- 应收账款增加 → 收入确认但未收款

**利润表 → 现金流量表**
- 净利润 → 经营活动现金流起点
- 折旧摊销 → 加回（非现金支出）
- 营运资本变动 → 调整项

**资产负债表 → 现金流量表**
- 固定资产增加 → 投资活动现金流流出
- 借款增加 → 筹资活动现金流流入
- 分红 → 筹资活动现金流流出

#### 3.2 财务健康度

| 指标 | 公式 | 健康标准 | 风险信号 |
|------|------|---------|---------|
| 资产负债率 | 总负债/总资产 | < 60% | > 70% |
| 流动比率 | 流动资产/流动负债 | > 1.5 | < 1 |
| 速动比率 | (流动资产 - 存货)/流动负债 | > 1 | < 0.8 |
| 有息负债率 | 有息负债/总资产 | < 40% | > 50% |
| 利息保障倍数 | EBIT/利息支出 | > 5 | < 2 |
| 经营性现金流 | 经营活动现金净流量 | 持续为正 | 持续为负 |

#### 3.3 盈利能力（杜邦分析）

```
ROE = 净利率 × 总资产周转率 × 权益乘数

高 ROE 的三种模式：
1. 高净利率型（茅台、苹果）- 产品溢价能力强
2. 高周转型（沃尔玛、亚马逊）- 运营效率高
3. 高杠杆型（银行、地产）- 财务杠杆高（风险较高）

可持续 ROE 判断：
- ROE > 15% 持续 5 年以上 = 优秀
- 需分析 ROE 来源是否可持续
- 警惕杠杆驱动的高 ROE
```

| 指标 | 公式 | 优秀标准 |
|------|------|---------|
| ROE | 净利润/净资产 | > 15% |
| ROA | 净利润/总资产 | > 8% |
| 毛利率 | 毛利/收入 | 行业前 30% |
| 净利率 | 净利润/收入 | 行业前 30% |
| ROIC | 息前税后利润/投入资本 | > WACC |

#### 3.4 成长能力

| 指标 | 公式 | 优秀标准 |
|------|------|---------|
| 营收增长率 | (本期 - 上期)/上期 | > 15% |
| 净利润增长率 | (本期 - 上期)/上期 | > 15% |
| 经营性现金流增长率 | (本期 - 上期)/上期 | > 净利润增长率 |
| 可持续增长率 | ROE × (1 - 分红率) | 与实际增长匹配 |

#### 3.5 营运能力

| 指标 | 公式 | 优秀标准 |
|------|------|---------|
| 应收账款周转率 | 收入/平均应收账款 | 行业前 30% |
| 存货周转率 | 成本/平均存货 | 行业前 30% |
| 总资产周转率 | 收入/平均总资产 | 行业前 30% |
| 现金循环周期 | 存货周转天数 + 应收周转天数 - 应付周转天数 | 越短越好 |

#### 3.6 ⚠️ 财务操纵识别清单（红色警报）

**必须检查的危险信号**：
- [ ] 应收账款增速 > 收入增速（连续 2 年）
- [ ] 存货增速 > 收入增速（连续 2 年）
- [ ] 经营性现金流持续 < 净利润（连续 3 年）
- [ ] 频繁变更会计政策或会计估计
- [ ] 关联交易占比 > 30%
- [ ] 毛利率显著高于同行且无合理解释
- [ ] 研发费用资本化比例 > 50%
- [ ] 其他应收款/其他应付款异常增长
- [ ] 在建工程长期不转固
- [ ] 商誉占净资产比例 > 30%

**发现 3 项以上红色警报 → 高度警惕，建议放弃研究**

- 历史言行一致性

### 5️⃣ 估值建模（1-2 天）⭐ 核心能力

#### 5.1 相对估值法

| 估值指标 | 公式 | 适用行业 | 注意事项 |
|---------|------|---------|---------|
| PE | 市值/净利润 | 稳定盈利行业 | 警惕周期性、一次性损益 |
| PB | 市值/净资产 | 重资产、金融 | 关注 ROE 水平 |
| PS | 市值/收入 | 高增长未盈利 | 需结合毛利率分析 |
| EV/EBITDA | 企业价值/息税折旧摊销前利润 | 资本密集型 | 排除折旧政策影响 |
| PEG | PE/净利润增长率 | 成长股 | 增长率预测准确性关键 |

**相对估值步骤**：
1. 选择 3-5 家可比公司
2. 计算各估值指标
3. 对比行业平均水平
4. 给予合理溢价/折价
5. 得出估值区间

#### 5.2 绝对估值法（DCF 模型）

**自由现金流计算**：
```
FCF = EBIT × (1 - 税率) + 折旧摊销 - 资本开支 - 营运资本增加

或：
FCF = 经营活动现金流 - 资本开支
```

**DCF 公式**：
```
企业价值 = ∑[FCFt / (1+WACC)^t] + 终值 / (1+WACC)^n

终值 = FCFn × (1 + g) / (WACC - g)

股权价值 = 企业价值 - 净债务
每股价值 = 股权价值 / 总股本
```

**关键假设参考**：
| 参数 | 取值原则 | 敏感性 |
|------|---------|-------|
| 收入增长率 | 参考历史增速 + 行业前景 | ±1% → 估值±10-15% |
| 永续增长率 g | 2-3%（不超过长期 GDP 增速） | ±0.5% → 估值±5% |
| WACC | 8-12%（视行业风险） | ±0.5% → 估值±5-10% |
| 预测期 | 5-10 年 | - |

**WACC 计算**：
```
WACC = (E/V) × Re + (D/V) × Rd × (1 - Tc)

其中：
E = 股权市值
D = 债务市值
V = E + D
Re = 股权成本 = Rf + β × (Rm - Rf)
   Rf = 无风险利率（10 年期国债）
   β = 个股贝塔系数
   Rm = 市场预期回报率
Rd = 债务成本（平均借款利率）
Tc = 企业所得税率
```

#### 5.3 估值结论

- **合理价值区间**: DCF 结果 ±20%
- **目标价位**: 合理价值下限（保守原则）
- **安全边际**: (内在价值 - 当前价格) / 内在价值
  - > 30%: 深度价值
  - 20-30%: 有吸引力
  - 10-20%: 合理
  - < 10%: 高估

### 6️⃣ 风险评估

#### 公司特定风险
- 客户集中风险（单一客户 > 30%）
- 供应商集中风险（单一供应商 > 30%）
- 产品单一风险（单一产品 > 50%）
- 管理层风险（核心人物依赖）
- 财务风险（高负债、现金流紧张）

#### 行业风险
- 政策风险（监管变化、补贴退坡）
- 竞争风险（新进入者、价格战）
- 技术风险（技术迭代、颠覆性创新）
- 需求风险（经济周期、消费习惯变化）

#### 市场风险
- 估值风险（市场情绪过热）
- 流动性风险（成交量低、机构持仓集中）
- 系统性风险（宏观经济、黑天鹅事件）

**风险等级评估**：
- 🟢 低风险：各项指标健康，风险可控
- 🟡 中风险：存在部分风险因素，需持续跟踪
- 🔴 高风险：多项风险警报，建议回避
