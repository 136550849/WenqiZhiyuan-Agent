## 🎯 核心投资原则

### 价值投资四原则
1. **护城河**: 只投资有持续竞争优势的企业
2. **安全边际**: 价格显著低于内在价值时买入
3. **能力圈**: 只投资自己理解的行业
4. **长期主义**: 忽略短期波动，持有优秀企业

### 财务分析核心
- ROE > 15% 持续 5 年 = 优秀企业标志
- 经营性现金流应持续 > 净利润
- 警惕应收账款/存货增速持续 > 收入增速

### ROE 杜邦分析
```
ROE = 净利率 × 总资产周转率 × 权益乘数

高 ROE 三种模式：
1. 高净利率型（茅台、苹果）- 产品溢价能力强
2. 高周转型（沃尔玛、亚马逊）- 运营效率高
3. 高杠杆型（银行、地产）- 财务杠杆高（风险较高）
```

### 三张表勾稽关系
- **利润表→资产负债表**: 净利润→未分配利润，折旧→固定资产减少
- **利润表→现金流量表**: 净利润为起点，折旧加回，营运资本调整
- **资产负债表→现金流量表**: 固定资产增加→投资流出，借款增加→筹资流入

### 估值纪律
- DCF 估值做敏感性分析（±1% 假设变化）
- 相对估值至少用 3 种方法交叉验证
- 安全边际 < 20% 不建议买入

### 估值方法要点
| 方法 | 适用场景 | 关键假设敏感性 |
|------|---------|---------------|
| PE | 稳定盈利行业 | 警惕周期性、一次性损益 |
| PB | 重资产、金融 | 需结合 ROE 水平 |
| PS | 高增长未盈利 | 需结合毛利率分析 |
| EV/EBITDA | 资本密集型 | 排除折旧政策影响 |
| DCF | 全行业 | 增长率±1% → 估值±10-15%，WACC±0.5% → 估值±5-10% |

**DCF 关键公式**:
```
FCF = EBIT×(1-税率) + 折旧摊销 - 资本开支 - 营运资本增加
WACC = (E/V)×Re + (D/V)×Rd×(1-Tc)
永续增长率 g = 2-3%（不超过长期 GDP 增速）
```

## 📋 研究流程标准（6 步法）

**完整研究周期**: 5-8 天

### 1. 信息收集（1-2 天）
- 年报/季报（3-5 年）
- 招股书/公告
- 券商研报（3-5 家）
- 行业数据

### 2. 业务分析（1-2 天）
- 收入结构拆解
- 商业模式画布
- 护城河评估（5 类）
- 竞争格局分析

### 3. 财务分析（1-2 天）⭐ 核心
- 三张表勾稽检查
- 杜邦分析
- 财务健康度评估
- 操纵风险排查（10 项红色警报）

### 4. 管理层评估（0.5 天）
- 背景调查
- 持股与激励
- 历史言行一致性
- 资本配置记录

### 5. 估值建模（1-2 天）⭐ 核心
- 相对估值（3-5 种方法）
- DCF 绝对估值
- 敏感性分析
- 安全边际计算

### 6. 报告撰写（1 天）
- 执行摘要
- 核心逻辑
- 风险提示
- 投资建议

## ⚠️ 财务操纵红色警报

**发现 3 项以上应高度警惕，建议放弃研究**：

1. 应收账款增速 > 收入增速（连续 2 年）
2. 存货增速 > 收入增速（连续 2 年）
3. 经营性现金流 < 净利润（连续 3 年）
4. 频繁变更会计政策或会计估计
5. 关联交易占比 > 30%
6. 毛利率显著高于同行无合理解释
7. 研发费用资本化比例 > 50%
8. 其他应收款/其他应付款异常增长
9. 在建工程长期不转固
10. 商誉占净资产比例 > 30%

## 🏰 护城河理论（巴菲特体系）

**五类护城河**：
1. **无形资产**: 品牌、专利、牌照、特许经营权（⭐⭐⭐⭐⭐）
2. **成本优势**: 规模效应、工艺优势、资源优势（⭐⭐⭐⭐）
3. **网络效应**: 用户越多价值越大（⭐⭐⭐⭐⭐）
4. **转换成本**: 客户更换供应商成本高（⭐⭐⭐⭐）
5. **规模优势**: 市场份额领先、固定成本摊薄（⭐⭐⭐）
