### 步骤 3：多周期分析

#### 月线周期（长期趋势）
```

判断：

- 月线下降 → 长期熊市
- 月线震荡 → 长期盘整
```

#### 周线周期（中期趋势）
```
分析内容：
- 20 周均线位置
- 周线级别趋势

判断：

- 周线下降 → 中期熊市
- 周线震荡 → 中期盘整
```

#### 日线周期（短期趋势）
```
分析内容：
- 20 日均线位置

判断：
- 日线上升 → 短期牛市
- 日线下降 → 短期熊市
- 日线震荡 → 短期盘整
```

#### 多周期共振
```
共振类型：
- 三线共振（月周日均上升）→ 强烈看涨 ⭐⭐⭐⭐⭐
- 二线共振（周日上升）→ 看涨 ⭐⭐⭐⭐
- 周期冲突（月升日降）→ 观望 ⭐⭐

操作策略：
- 共振看涨 → 重仓参与
- 共振看跌 → 空仓观望
- 周期冲突 → 轻仓试探
```

### 步骤 4：K 线形态深度分析

#### 单 K 线形态（20+ 种）
```
反转形态：
1. 锤头线（Hammer）
   - 特征：下影线≥实体 2 倍，上影线很短
   - 位置：下降趋势底部
   - 信号：底部反转

2. 吊颈线（Hanging Man）
   - 特征：同锤头线
   - 位置：上升趋势顶部
   - 信号：顶部反转

3. 倒锤头（Inverted Hammer）
   - 特征：上影线≥实体 2 倍
   - 位置：下降趋势底部
   - 信号：底部反转

4. 射击之星（Shooting Star）
   - 特征：同倒锤头
   - 位置：上升趋势顶部
   - 信号：顶部反转

5. 十字星（Doji）
   - 特征：实体<1%
   - 位置：任何位置
   - 信号：变盘信号

持续形态：
6. 大阳线（Marubozu）
   - 特征：实体>3%，无上下影线
   - 信号：强烈看涨

7. 大阴线
   - 特征：实体>3%，无上下影线
   - 信号：强烈看跌

...（共 20+ 种）
```

### 步骤 5：技术指标深度分析（10+ 指标）

#### 趋势指标（4 个）
```
1. MACD（平滑异同移动平均线）
   分析维度：
   - DIF 与 DEA 位置（0 轴上方/下方）
   - 金叉/死叉
   - 顶背离/底背离
   - 红绿柱变化
   
   信号强度：
   - 0 轴上方金叉 + 背离 → ⭐⭐⭐⭐⭐
   - 0 轴上方金叉 → ⭐⭐⭐⭐
   - 0 轴下方金叉 → ⭐⭐⭐
   - 单纯金叉/死叉 → ⭐⭐

2. DMI（趋向指标）
   分析维度：
   - +DI 与-DI 交叉
   - ADX 数值（趋势强度）
   - ADX 方向
   
   判断：
   - +DI 上穿-DI + ADX>25 → 强烈上涨趋势
   - -DI 上穿+DI + ADX>25 → 强烈下跌趋势
   - ADX<20 → 无趋势，震荡

3. MA（移动平均线）
   分析维度：
   - 均线排列（多头/空头）
   - 均线交叉（金叉/死叉）
   - 均线支撑/阻力
   
   均线系统：
   - 5 日、10 日、20 日（短期）
   - 60 日（中期）
   - 120 日、200 日（长期）
   
   多头排列：5>10>20>60>120>200 → 强烈看涨
   空头排列：5<10<20<60<120<200 → 强烈看跌

4. EMA（指数平均线）
   特点：对近期价格更敏感
   应用：短线交易、快速反应
```

#### 动能指标（4 个）
```
1. RSI（相对强弱指标）
   分析维度：
   - 数值区间（0-100）
   - 超买/超卖（>70/<30）
   - 背离信号
   - 趋势线突破
   
   进阶用法：
   - RSI 50 分界线（多空分界）
   - RSI 形态（头肩顶/底）
   - RSI 失败摆动

2. KDJ（随机指标）
   分析维度：
   - K、D、J 三线位置
   - 金叉/死叉
   - 超买/超卖（>80/<20）
   - 钝化现象
   
   J 线特殊用法：
   - J>100 → 严重超买
   - J<0 → 严重超卖
   - J 线背离 → 强烈信号

3. Williams %R（威廉指标）
   分析维度：
   - 数值区间（0 到 -100）
   - 超买/超卖（>-20/<-80）
   
   特点：反应灵敏，适合短线

4. CCI（商品通道指标）
   分析维度：
   - 数值区间
   - 超买/超卖（>100/<-100）
   - 突破信号
   
   特点：适合震荡市
```

#### 波动指标（3 个）
```
1. 布林带（Bollinger Bands）
   分析维度：
   - 价格位置（上轨/中轨/下轨）
   - 带宽变化（扩张/收缩）
   - 突破信号
   - 布林带挤压
   
   进阶用法：
   - 布林带带宽<5% → 即将变盘
   - 价格连续触及上轨 → 强势上涨
   - 价格连续触及下轨 → 强势下跌

2. ATR（平均真实波幅）
   分析维度：
   - ATR 数值（波动率）
   - ATR 趋势（上升/下降）
   
   应用：
   - 止损位设置（2×ATR）
   - 仓位管理（波动大仓位小）

3. 标准差（Standard Deviation）
   分析维度：
   - 价格离散程度
   - 波动率变化
   
   应用：风险评估
```

#### 成交量指标（3 个）
```
1. OBV（能量潮）
   分析维度：
   - OBV 趋势
   - OBV 与价格背离
   
   信号：
   - 价格涨 + OBV 涨 → 健康上涨
   - 价格涨 + OBV 跌 → 上涨乏力
   - OBV 背离 → 变盘信号

2. 成交量均线
   分析维度：
   - 5 日量均线
   - 10 日量均线
   - 量均线交叉
   
   信号：
   - 成交量上穿量均线 → 放量
   - 成交量下穿量均线 → 缩量

3. 量比
   公式：量比 = 当日成交量 / 过去 5 日平均成交量
   
   判断：
   - 量比>3 → 异常放量
   - 量比 1.5-3 → 明显放量
   - 量比 0.8-1.5 → 正常
   - 量比<0.8 → 缩量
```

### 步骤 6：图表模式识别

#### 反转形态（6 种）
```
1. 头肩顶/底（Head and Shoulders）⭐⭐⭐⭐⭐
   - 可靠性：80%
   - 目标位：头部到颈线的垂直距离
   - 成交量：右肩成交量明显萎缩

2. 双顶/双底（Double Top/Bottom）⭐⭐⭐⭐
   - 可靠性：75%
   - 目标位：顶部/底部到颈线的距离
   - 确认：跌破/突破颈线

3. 三重顶/底（Triple Top/Bottom）⭐⭐⭐⭐
   - 可靠性：75%
   - 比双顶/底更可靠

4. 圆弧顶/底（Rounding Top/Bottom）⭐⭐⭐
   - 可靠性：65%
   - 形成时间长
   - 趋势反转缓慢

5. V 型顶/底（V Top/Bottom）⭐⭐⭐
   - 可靠性：60%
   - 反转突然
   - 难以捕捉

6. 岛型反转（Island Reversal）⭐⭐⭐⭐
   - 可靠性：75%
   - 特征：缺口 + 盘整 + 反向缺口
```

#### 持续形态（5 种）
```
1. 三角形（Triangle）
   - 对称三角形：中性，突破方向不定 ⭐⭐⭐
   - 上升三角形：看涨 ⭐⭐⭐⭐
   - 下降三角形：看跌 ⭐⭐⭐⭐
   - 目标位：三角形最宽处的高度

2. 旗形（Flag）
   - 上升旗形：看涨 continuation ⭐⭐⭐⭐
   - 下降旗形：看跌 continuation ⭐⭐⭐⭐
   - 目标位：旗杆高度

3. 三角旗形（Pennant）
   - 小型对称三角形
   - 趋势中途出现
   - 目标位：旗杆高度

4. 楔形（Wedge）
   - 上升楔形：看跌 ⭐⭐⭐⭐
   - 下降楔形：看涨 ⭐⭐⭐⭐
   - 与三角形区别：两边同向

5. 矩形（Rectangle）
   - 箱体震荡
   - 突破方向：通常延续原趋势
   - 目标位：箱体高度
```

### 步骤 7：支撑阻力深度分析

#### 支撑位识别（7 层）
```
优先级从高到低：
1. 前期低点（最近 3 个月）
2. 重要均线（200 日/120 日/60 日）
3. 缺口下沿
4. 成交密集区下沿
5. 整数关口
6. 黄金分割位（0.382/0.5/0.618）
7. 趋势线支撑

强度评估：
- 被测试次数越多，强度越弱
- 时间越近，强度越强
- 成交量越大，强度越强
```

#### 阻力位识别（7 层）
```
优先级从高到低：
1. 前期高点（最近 3 个月）
2. 重要均线（下行时）
3. 缺口上沿
4. 成交密集区上沿
5. 整数关口
6. 黄金分割位
7. 趋势线阻力
```

#### 支撑阻力转换
```
原理：支撑跌破后变阻力，阻力突破后变支撑

应用：
- 突破阻力位后，回踩确认是买点
- 跌破支撑位后，反抽确认是卖点
```

```
5. 价平量增 → 变盘前兆 ⭐⭐⭐⭐
6. 价平量缩 → 观望情绪 ⭐⭐
7. 价涨量平 → 温和上涨 ⭐⭐⭐⭐
8. 价跌量平 → 温和下跌 ⭐⭐⭐
```

#### 成交量形态
```
1. 成交量堆
   - 特征：成交量突然放大，形成小丘
   - 信号：主力进场/出场

2. 成交量天量
   - 特征：历史最大成交量
   - 位置：顶部→见顶信号；底部→见底信号

3. 成交量地量
   - 特征：历史最小成交量
   - 信号：市场极度低迷，可能见底

4. 成交量温和放大
   - 特征：成交量逐步增加
   - 信号：健康上涨
```
