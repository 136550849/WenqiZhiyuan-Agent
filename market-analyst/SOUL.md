### 步骤 3：趋势分析（三重时间框架）

#### 长期趋势（200 日均线）
```
判断标准：
- 价格在 200 日均线上方 → 长期上升趋势
- 价格在 200 日均线下方 → 长期下降趋势
- 200 日均线走平 → 长期震荡
```

#### 中期趋势（60 日均线）
```
判断标准：
- 60 日均线向上 → 中期上升趋势
- 60 日均线向下 → 中期下降趋势
- 60 日均线走平 → 中期震荡
```

#### 短期趋势（20 日均线）
```
判断标准：
- 20 日均线向上 → 短期上升趋势
- 20 日均线向下 → 短期下降趋势
- 20 日均线走平 → 短期震荡
```

### 步骤 5：技术指标分析

#### MACD（平滑异同移动平均线）
```
分析维度：
1. DIF 与 DEA 位置（0 轴上方/下方）
2. 金叉/死叉（DIF 上穿/下穿 DEA）
3. 背离（价格新高/新低，MACD 未新高/新低）
4. 红绿柱变化（动能强弱）

信号判断：
- 0 轴上方金叉 → 强烈买入信号
- 0 轴下方金叉 → 一般买入信号
- 0 轴上方死叉 → 一般卖出信号
- 0 轴下方死叉 → 强烈卖出信号
- 顶背离 → 看跌信号
- 底背离 → 看涨信号
```

#### RSI（相对强弱指标）
```
分析维度：
1. 数值区间（0-100）
2. 超买/超卖（>70/<30）
3. 背离信号
4. 趋势线突破

信号判断：
- RSI > 70 → 超买，警惕回调
- RSI < 30 → 超卖，关注反弹
- RSI 50-70 → 强势区间
- RSI 30-50 → 弱势区间
```

#### KDJ（随机指标）
```
分析维度：
1. K、D、J 三线位置
2. 金叉/死叉
3. 超买/超卖（>80/<20）
4. 钝化现象

信号判断：
- K>D 且<80 → 买入
- K<D 且>20 → 卖出
- J>100 → 严重超买
- J<0 → 严重超卖
```

#### 布林带（Bollinger Bands）
```
分析维度：
1. 价格位置（上轨/中轨/下轨）
2. 带宽变化（扩张/收缩）
3. 突破信号

信号判断：
- 触及上轨 → 可能回调
- 触及下轨 → 可能反弹
- 带宽收缩 → 即将变盘
- 突破上轨 → 强势上涨
- 跌破下轨 → 弱势下跌
```

#### 均线系统
```
常用均线：
- MA5：超短期，攻击线
- MA10：短期，操盘线
- MA20：中短期，月线
- MA60：中期，季线
- MA120：中长期，半年线
- MA200：长期，年线

信号判断：
- 多头排列：MA5>MA10>MA20>MA60 → 强势上涨
- 空头排列：MA5<MA10<MA20<MA60 → 强势下跌
- 金叉：短期均线上穿长期均线 → 买入信号
- 死叉：短期均线下穿长期均线 → 卖出信号
```

### 步骤 6：支撑阻力分析

#### 支撑位识别（5 层法）
```
优先级：
1. 前期低点（最强支撑）
2. 重要均线（60 日/120 日/200 日）
3. 缺口下沿
4. 整数关口（心理支撑）
5. 成交密集区下沿
```

#### 阻力位识别（5 层法）
```
优先级：
1. 前期高点（最强阻力）
2. 重要均线（下行时）
3. 缺口上沿
4. 整数关口（心理阻力）
5. 成交密集区上沿
```

### 步骤 7：成交量分析

#### 量价关系
| 价格 | 成交量 | 解读 |
|------|--------|------|
| 上涨 | 放量 | 健康上涨 |
| 上涨 | 缩量 | 上涨乏力 |
| 下跌 | 放量 | 恐慌下跌 |
| 下跌 | 缩量 | 下跌趋缓 |
| 横盘 | 缩量 | 观望情绪 |
| 横盘 | 放量 | 变盘前兆 |

#### 量比分析
```
量比 = 当日成交量 / 过去 5 日平均成交量

量比 > 3 → 异常放量
量比 1.5-3 → 明显放量
量比 0.8-1.5 → 正常
量比 < 0.8 → 缩量
```
