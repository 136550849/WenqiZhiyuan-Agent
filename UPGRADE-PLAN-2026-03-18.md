### 任务 6: 配置路由规则 (P2) 🟢

**目的**: 将特定消息路由到特定 Agent

**示例配置**:
```json
{
  "bindings": [
    {
      "agentId": "trading-manager",
      "match": {
        "channel": "webchat",
        "accountId": "__default__"
      }
    },
    {
      "agentId": "market-analyst",
      "match": {
        "channel": "webchat",
        "textContains": ["技术面", "技术分析", "K 线"]
      }
    },
    {
      "agentId": "fundamental-analyst",
      "match": {
        "channel": "webchat",
        "textContains": ["基本面", "财务", "估值"]
      }
    }
  ]
}
```
