## 📊 团队能力总览

### 核心能力矩阵

| 能力域 | 负责 Agent | 核心技能 |
|--------|------------|----------|
| **技术面** | market-analyst, technical-analyst | K 线、指标、图表模式 |
| **基本面** | fundamental-analyst, researcher-2 | 财务分析、估值建模 |
| **行业研究** | researcher-1 | 行业分析、竞争格局 |
| **消息面** | news-analyst | 新闻资讯、政策解读 |
| **情绪面** | sentiment-analyst | 投资者情绪、市场热度 |
| **风险管理** | risk-analyst | 风险识别、评估、控制 |
| **交易决策** | trading-manager, trader | 综合决策、交易计划 |
| **资产配置** | portfolio-manager | 组合构建、仓位管理 |
| **研究统筹** | research-lead | 任务分配、质量把控 |
| **报告输出** | post-processor | 报告格式化、美化 |

### 团队协作流程

```
用户请求
    ↓
trading-manager（接收任务）
    ↓
┌───────────────────────────────────────┐
│  并行分析                              │
│  ├─ market-analyst（技术面）          │
│  ├─ fundamental-analyst（基本面）     │
│  ├─ news-analyst（消息面）           │
│  ├─ sentiment-analyst（情绪面）      │
│  └─ risk-analyst（风险评估）         │
└───────────────────────────────────────┘
    ↓
trading-manager（综合判断）
    ↓
trader（制定交易计划）
    ↓
post-processor（生成报告）
    ↓
用户
```
